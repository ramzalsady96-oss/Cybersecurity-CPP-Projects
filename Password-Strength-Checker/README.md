# 🛡️ Password Strength Checker

A C++ application that evaluates password strength based on basic security rules.

## Features

- Password length validation
- Uppercase letter detection
- Number detection
- Symbol detection
- Weak / Medium / Strong evaluation

## Technologies

- C++
- Object-Oriented Programming (OOP)

## How to Compile

```bash
g++ main.cpp -o password-checker
```

## Run

```bash
./password-checker
```
