# 🛡️ C++ Cybersecurity Projects

A collection of beginner-friendly cybersecurity projects developed in **C++** as part of my cybersecurity learning journey.

---

## 📂 Projects

### 🔐 Caesar Cipher
A simple implementation of the Caesar Cipher algorithm for encrypting and decrypting text.

**Features**
- Encrypt text
- Decrypt text
- Adjustable encryption key
- Object-Oriented Programming (OOP)

---

### 🔑 Password Generator
A simple password generator that creates stronger passwords using random numbers and special characters.

**Features**
- Custom username
- Random numbers
- Special characters

---

### 🛡️ Password Strength Checker
Checks the strength of a password based on basic security rules.

**Features**
- Password length check
- Uppercase letters
- Numbers
- Symbols
- Weak / Medium / Strong evaluation

---

### 🔒 Secure Login System
A basic authentication system with protection against brute-force login attempts.

**Features**
- Username & password authentication
- Three login attempts
- Account lock
- Login report

---

### 📊 Security Log Analyzer
Analyzes login events and reports suspicious activity.

**Features**
- Count successful logins
- Count failed logins
- Display security report
- Warn about multiple failed attempts
